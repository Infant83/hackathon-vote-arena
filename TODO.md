## 현재 상태 요약

- 운영 주소: `https://hack.lgdisplay.workers.dev`
- 운영 Worker: `hack`
- 기본 운영 화면:
  - `/vote`: 관객 투표/응원/퀴즈 참여 화면
  - `/wall`: 발표장 송출 화면
  - `/admin`: 관리자 운영 콘솔
  - `/admin?showCheer=1`: 응원 메시지 Showup
- 로컬 검증 기준:
  - `npm run lint`
  - `npm run build`
  - 실제 상태 공유 검증은 Vite dev server가 아니라 `npm run realtime` 또는 Cloudflare Worker에서 수행

  - 운영 상태는 Durable Object storage에 저장하고, 필요한 경우 JSON으로 내려받아 백업한다.

- [x] 이번 행사 전에는 D1/외부 DB 마이그레이션을 하지 않는다.
  - Durable Object storage를 운영 저장소로 사용하고, JSON/XLSX 내보내기로 백업한다.
- [x] 이번 행사 전에는 커스텀 도메인을 추가하지 않는다.
